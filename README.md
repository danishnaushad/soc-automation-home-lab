# SOC Automation Home Lab

## Overview
This project demonstrates a SOC (Security Operations Center) Home Lab using Wazuh SIEM for monitoring, detection, and security analysis.

## Features
- Wazuh SIEM Setup
- File Integrity Monitoring (FIM)
- Windows Agent Monitoring
- Ubuntu Wazuh Manager
- Security Event Collection
- Real-Time Alerting

## Lab Architecture
- Ubuntu Server (Wazuh Manager)
- Windows Host (Wazuh Agent)
- VirtualBox Bridged Networking

## Technologies Used
- Wazuh
- Ubuntu Server
- Windows
- VirtualBox
- Linux
- Networking

## Current Status
Project currently in progress. More configurations, screenshots, alerts, and automation scripts will be uploaded soon.

## Goals
- Improve SOC monitoring skills
- Understand SIEM workflows
- Practice threat detection and log analysis
- Build hands-on cybersecurity experience
